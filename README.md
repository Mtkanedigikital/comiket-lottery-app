# distribution-lottery-app

このリポジトリは、頒布会などのイベントで使用する「抽選・配布管理システム」です。  
フロントエンドは React、バックエンドは Node.js (Express) で構成されています。

---

## 🚀 Features
- 抽選番号の管理
- 当選結果の照会
- 配布物の管理

---

## 📦 Setup

### Frontend
```bash
echo "# Setup frontend"
cd frontend
npm install
npm start
```

### Backend
```bash
echo "# Setup backend"
cd backend
npm install
npm start
```
